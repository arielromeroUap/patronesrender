Server-Side Rendering (SSR)

"El plato llega ya cocinado"

💻 Request ➜ Servidor Node ➜ HTML + Datos

Pros

SEO excelente, carga inicial rápida.

Contras

Mayor carga en el CPU del servidor.

Static Site Generation (SSG)

Build Time

Se generan los HTML físicos una sola vez al compilar.

$ npm run build
Creating 1500 static pages...

Runtime

El usuario descarga un archivo de una CDN.

¿Qué patrón elegir?

Dashboard / Apps Privadas: Client-Side Rendering (CSR).
E-commerce / Perfiles: Server-Side Rendering (SSR).
Blogs / Landing Pages: Static Site Generation (SSG).
